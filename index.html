Javascript Project

Run Sonic

Run Sonic is a video made with only javascript. Initially, Doctor Eggman is chasing Sonic. In order to help Sonic run away from Doctor Eggman, player has to type corresponding falling letter showing on the screen as fast as he or she can. The game is over when Doctor Eggman catches Sonic which means the player did not type right letter fast enough. To play the game, please click the image above.
